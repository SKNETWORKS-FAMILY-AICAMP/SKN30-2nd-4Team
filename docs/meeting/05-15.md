---

### 1. 논의 내용

* **모델 방향:** 과거 상영작 데이터를 통한 개봉 예정작 흥행 예측 (머신러닝 활용)
* **Feature 추가:** 배우 및 감독 파워 지표 반영 필요
* **학습 범위 제한:** 데이터 왜곡 방지를 위해 **2025년 10월 개봉작**까지만 학습 (최소 2개월 데이터 확보를 위해 12월까지 수집)
* **데이터 관리:** DB 원본은 유지하되, 학습 시 SQL로 Feature 조회 후 CSV로 추출
* **수집 기준:** 박스오피스 상위 20개 위주

---

### 2. 개발 및 데이터 수집 흐름

1. **Step 1:** API 호출 및 일별 박스오피스 데이터 수집
2. **Step 2:** 영화사·영화인 코드 추출 및 상세 정보 수집
3. **Step 3:** SQL을 사용한 Feature 추출 및 CSV 저장
4. **Step 4:** 도출된 Feature 기반 모델 학습 및 성능 평가/개선

---

### 3. 모델 설계 (Table & Model)

#### **데이터베이스 테이블**

* `DAILY_BOX_OFFICE`: KOBIS API 기반 영화 기본 정보
* `MOVIES`: 영화 상세 메타데이터 및 JSON 원본
* `MARKET_DAILY`: 일별 시장 집계(총 관객수, 총 상영횟수 등)

#### **분석 모델**

| 구분 | 회귀 (Regression) | 분류 (Classification) |
| --- | --- | --- |
| **예측값** | 누적 관객수 | 흥행 성공 / 보통 / 실패 |
| **추천 모델** | XGBoost, LightGBM, CatBoost | Random Forest, XGBoost, Logistic Regression |
